**Расположение решения в пакете:**

## Комментарии по решению.

- Чтобы минимизировать код в main(), вынес всё, что связано с двумерным массивом (далее - "матрица") в отдельный класс - `Matrix`;
- Создал поле - `int[][] COLORS`, сделал его `private final` так как мы уже изначально знаем его размерность и менять не планируем;
- Инициализируем матрицу непосредственно в конструкторе, используя `private` метод `matrixInit()`, и выводим на экран.
- Вывод матрицы - `public void getMatrix();`
- Поворот матрицы по часовой стрелке на 90° - `public int[][] turn90ClockWise()`
- Поворот матрицы по часовой стрелке на 180° - `public int[][] turn180ClockWise()`
- Поворот матрицы по часовой стрелке на 270° - `public int[][] turn270ClockWise()`
