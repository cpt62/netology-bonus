# Решение практического занятия "Многомерные массивы"

**Расположение решения в пакете:**  
/src/ru/netology/homeworks/basic/task6

## Комментарии по решению.

- Чтобы минимизировать код в main(), вынес всё, что связано с двумерным массивом (далее - "матрица") в отдельный класс - `Matrix`;
- Создал `private final` поле - `int[][] COLORS`, так как мы уже изначально знаем его размерность.
- Инициализирую матрицу непосредственно в конструкторе, используя `private` метод `matrixInit()`
- Вывод матрицы - `public void getMatrix();`
