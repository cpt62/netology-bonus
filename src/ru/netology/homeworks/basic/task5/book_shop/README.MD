**Расположение решения в пакете:**

## Комментарии по решению.

- Предоставляю решение, включая Задание 2 (НЕобязательное)
- Поля классов сделал с модификатором доступа `private` + `setters/getters`
- Для демонстрации работы методов согласно задания, создал отдельный метод `void bookInfo(String searchString)` в классе `Book`. 
На вход принимает параметр для поиска подстроки в названии книги/имени автора/фамилии автора.
- С целью демонстрации работы методов, создал класс `Main`, чтобы было нагляднее, без нагромождения.
